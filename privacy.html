Your files never leave your device.

Analyser reads and analyses every file entirely in your browser. There is no upload and no server that receives it - the name, contents and bytes of anything you drop stay with you, and are gone the moment you close the tab.

What the two counters do.

Two anonymous, aggregate counters, shown on the stats page. Neither can identify you or your files.

Files analysed
Analysing a file sends only its lowercased extension - say jpg or sldprt - to a tally. Never the name, size or contents. Unrecognised types are counted too, as an honest wish-list of what to add next.
Visitors
A simple, anonymous count of visits - no accounts, no cookies, and nothing that identifies you.

The small print, kept small.

No third-party analytics, ads or trackers - the counters run on the site’s own Cloudflare domain. They are best-effort and approximate: a count is skipped if a request fails or you are offline, and nothing is sent at all when you use Analyser offline or as an installed app. Like any website, the host sees basic request data such as IP addresses to serve pages; Analyser keeps only the anonymous counts above, nothing that identifies you.

Dependencies
Metadata
exifr. EXIF · IPTC · XMP · ICC.
OCR
tesseract.js. On-device text recognition with language picker.
QR codes
jsQR. Scans for QR codes in photos. Loaded only when needed.
Map
Leaflet with OpenStreetMap tiles. Loaded only if the photo has GPS.
HEIC
heic2any. Converts HEIC/HEIF to JPEG in the browser.
RAW
magick-wasm. ImageMagick compiled to WASM. Converts ARW, CR2, CR3, NEF, and other RAW formats. ~15 MB, loaded only when needed.
Audio
Native Web Audio API plus a hand-written radix-2 FFT for spectrogram and analysis. BPM read from ID3/MP4 tags when available, estimated via onset detection otherwise.
Video
HTML5 video for playback. Frame rate parsed from MP4 container metadata with FFmpeg fallback. Automatic scene detection with confidence scoring. iOS visible-player fallback.
FFmpeg
ffmpeg.wasm. Audio extraction and frame rate detection for formats the browser can't decode natively. The ~31 MB core is fetched from a CDN the first time it's needed (then cached for offline use).
PDF
pdf.js by Mozilla. Renders PDF pages, extracts text, and supports OCR scanning of image-only pages via Tesseract. Loaded only for PDF files.
3D / CAD
occt-import-js (OpenCASCADE). Tessellates STEP, IGES and BREP CAD solids to meshes for the in-browser 3D viewer. ~9 MB, in the Everything download (or loaded on demand).
Archives
fflate for ZIP, libarchive.js for RAR / 7-Zip (CBR / CB7 comics), plus fzstd (Zstandard) and xzwasm (XZ / LZMA2) decompressors. Loaded on demand.
Databases
sql.js. SQLite compiled to WASM - lists tables, schema and row counts from .sqlite / .db files.
JPEG 2000
OpenJPEG compiled to WASM. Decodes JP2 / J2K images the browser can’t. Loaded only when needed.
PostScript
Ghostscript compiled to WASM. Renders EPS / PostScript graphics. ~16 MB, in the Complete download only.
Animation
lottie-web. Plays Lottie / Bodymovin JSON animations. Loaded only when needed.
File ID
Built-in identification for 1017+ file formats. Adobe, SolidWorks, Fusion 360, AutoCAD, Blender, Office, 3D mesh, executables, fonts, G-Code, log files, and more. Extracts version, author, and metadata from headers, ZIP manifests, ID3 tags, and gzipped XML projects. DOCX documents render with formatted text. AI-generation markers detected in photo metadata.
Install
Add to home screen for an offline PWA. Service worker caches the app shell. Use the buttons above to pre-cache heavy dependencies.
