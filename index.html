Supported formats

01 Photo

Read every byte of metadata a camera left behind.

EXIF, IPTC, XMP, ICC. Dimensions, exact aspect ratio, RGB histogram, dominant palette, GPS map, on-device OCR.

02 Sound

Inspect the spectrum of any audio source.

Open a file, record from the mic, or visualise it live. Linear or logarithmic axis, adjustable FFT size and window, choice of colourmap, save as PNG, fullscreen.

03 Video

Decode every frame and frequency in a video.

Container and codec detection, resolution, frame rate, bitrate. Capture any frame for full photo analysis. Extract the audio track for waveform and spectrogram.

Everything runs in your browser.

No upload, no analytics, no servers in the loop.

Dependencies
Metadata
exifr. EXIF · IPTC · XMP · ICC.
OCR
tesseract.js. On-device text recognition with language picker.
QR codes
jsQR. Scans for QR codes in photos. Loaded only when needed.
Map
Leaflet with OpenStreetMap tiles. Loaded only if the photo has GPS.
HEIC
heic2any. Converts HEIC/HEIF to JPEG in the browser.
RAW
magick-wasm. ImageMagick compiled to WASM. Converts ARW, CR2, CR3, NEF, and other RAW formats. ~15 MB, loaded only when needed.
Audio
Native Web Audio API plus a hand-written radix-2 FFT for spectrogram and analysis. BPM read from ID3/MP4 tags when available, estimated via onset detection otherwise.
Video
HTML5 video for playback. Frame rate parsed from MP4 container metadata with FFmpeg fallback. Automatic scene detection with confidence scoring. iOS visible-player fallback.
FFmpeg
ffmpeg.wasm. Audio extraction and frame rate detection for formats the browser can't decode natively. ~30 MB, loaded only when needed.
PDF
pdf.js by Mozilla. Renders PDF pages, extracts text, and supports OCR scanning of image-only pages via Tesseract. Loaded only for PDF files.
Archives
fflate. Fast ZIP decompression in the browser.
File ID
Built-in identification for 200+ file formats. Adobe, SolidWorks, Fusion 360, AutoCAD, Blender, Office, 3D mesh, executables, fonts, G-Code, log files, and more. Extracts version, author, and metadata from headers, ZIP manifests, ID3 tags, and gzipped XML projects. DOCX documents render with formatted text. AI-generation markers detected in photo metadata.
Install
Add to home screen for an offline PWA. Service worker caches the app shell. Use the buttons above to pre-cache heavy dependencies.
