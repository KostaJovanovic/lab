01 Photo

Read every byte of metadata a camera left behind.

JPEG, HEIC, PNG, WEBP, RAW. EXIF, IPTC, XMP, ICC. Dimensions, exact aspect ratio, RGB histogram, dominant palette, GPS map, on-device OCR.

02 Sound

Inspect the spectrum of any audio source.

Open a file, record from the mic, or visualise it live. Linear or logarithmic axis, adjustable FFT size and window, choice of colourmap, save as PNG, fullscreen.

03 Video

Decode every frame and frequency in a video.

Container and codec detection, resolution, frame rate, bitrate. Capture any frame for full photo analysis. Extract the audio track for waveform and spectrogram.

04 About

Everything runs in your browser.

Files are read on your device. No upload, no analytics, no servers in the loop.

Metadata
exifr — EXIF · IPTC · XMP · ICC.
OCR
tesseract.js — auto runs English + Serbian (Latin + Cyrillic). Picker for other languages.
Map
Leaflet with OpenStreetMap tiles. Loaded only if the photo has GPS.
Audio
Native Web Audio API plus a hand-written radix-2 FFT.
Video
HTML5 video for playback and frame capture. Audio extracted via Web Audio API for spectrogram.
HEIC
heic2any — converts HEIC/HEIF to JPEG in your browser. Loaded only when needed.
Install
Add to home screen for an offline PWA.
