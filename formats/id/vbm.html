What is a .VBM file?

VBM is a recorded input movie from the VisualBoyAdvance emulator.

Did you know
  • VBM replays a Game Boy game by replaying every recorded button press.
  • A VBM is recorded by VBA-ReRecording, a special build of VisualBoyAdvance used by tool-assisted speedrunners for frame-perfect, deterministic playback.
  • After its header a VBM reserves 64 bytes for the author’s name and 128 bytes for a description of the recorded run.
What Analyser reads
Identify and read more game and emulator files: Unity asset bundles, Unreal cooked assets, CISO/CHD disc images, FMOD/Wwise sound banks, Spine skeletons and atlases, GameMaker/LDtk/TIC-80 projects, Minecraft Anvil regions, Warcraft III maps, Ren Py and RPG Maker data, extra console ROMs (Atari, PC Engine, Master System/Game Gear, WonderSwan, 3DS homebrew), xdelta/Basis patches and emulator saves and movies.
Depth of analysis
.VBM is an identification-grade format: Analyser recognises it from its bytes and decodes the header metadata it carries, rather than opening it in a full viewer. Formats that do get a full viewer are marked "Full" on the formats page.
Open a .VBM file
Drag a .VBM file onto the Analyser home page (or tap to pick one). It is identified entirely in your browser - nothing is uploaded, there is no account, and it works offline once installed.
