i Why

Why even waste time on this, aren't there many similar tools?

No. All the tools I have found are either full of ads, require uploading, or even worse: Ugly, unresponsive and unintuitive.
I have made this tool primarily for my needs, but it spiraled out of control and became whatever you are seeing right now.
I was unable to find a satisfactory spectrogram app / website / program, since all of them are either far too complex for my liking and needs, don't work as they should, or require a multiple gigabyte installation. No more.

This tool has a bunch of functions that I needed, or anticipate that I will need in the future, and equal number of ones that I don't or never will.

ii Where

Works without internet.

Analyser is a Progressive Web App. Install it to your home screen or desktop and it works offline. Use the download buttons on the main page to pre-cache dependencies, from the essentials (~48 MB) up to all OCR languages (~310 MB; the larger packs stream from a CDN).

The service worker caches the entire app shell on first visit. Heavy tools like FFmpeg, ImageMagick, and Tesseract are cached separately when you download them, so the initial load stays light.

iii What

A forensic tool that runs entirely in your browser.

Analyser reads photos, sound, video, PDFs, archives, and over 1,000 file formats. It extracts every byte of metadata, visualises spectra, decodes frames, and identifies files, all without uploading a single bit.

Photo
EXIF, IPTC, XMP, ICC. Dimensions, aspect ratio, RGB histogram, dominant palette, GPS map, QR code detection, OCR. Supports JPEG, PNG, WebP, HEIC, and RAW formats (ARW, CR2, CR3, NEF, etc.).
Sound
Waveform, spectrogram (linear or logarithmic), adjustable FFT size and window function, colourmap selection, fullscreen mode. Open a file, record from the mic, or visualise live input. MP3, WAV, M4A, FLAC, OGG, OPUS.
Video
Container and codec detection, resolution, frame rate (parsed from MP4 container with FFmpeg fallback), bitrate, duration. Frame-by-frame navigation with editable timecode and an in-app AVI player with synced audio you can mute. Capture any frame for full photo analysis. Extract the audio track for waveform and spectrogram, and map editing timelines (EDL, FCPXML, OTIO).
PDF
Metadata, page count, dimensions. Full text extraction with pagination. Page previews. OCR scanning of image-only pages via Tesseract.
Archives
ZIP file listing with sizes, dates, and compression ratios.
1017+ formats
Adobe (PSD, AI, INDD), AutoCAD (DWG, DXF), SolidWorks, Fusion 360, Blender, 3D meshes (FBX, GLB, STL, OBJ, 3MF, STEP, IGES), Office documents (DOCX, XLSX, PPTX), executables (EXE, DLL), fonts, game assets, and web files (HTML, CSS, JS), and more. Header parsing extracts metadata where possible.

Dependencies
Metadata
exifr. EXIF · IPTC · XMP · ICC.
OCR
tesseract.js. On-device text recognition with language picker.
QR codes
jsQR. Scans for QR codes in photos. Loaded only when needed.
Map
Leaflet with OpenStreetMap tiles. Loaded only if the photo has GPS.
HEIC
heic2any. Converts HEIC/HEIF to JPEG in the browser.
RAW
magick-wasm. ImageMagick compiled to WASM. Converts ARW, CR2, CR3, NEF, and other RAW formats. ~15 MB, loaded only when needed.
Audio
Native Web Audio API plus a hand-written radix-2 FFT for spectrogram and analysis. BPM read from ID3/MP4 tags when available, estimated via onset detection otherwise.
Video
HTML5 video for playback. Frame rate parsed from MP4 container metadata with FFmpeg fallback. Automatic scene detection with confidence scoring. iOS visible-player fallback.
FFmpeg
ffmpeg.wasm. Audio extraction and frame rate detection for formats the browser can't decode natively. The ~31 MB core is fetched from a CDN the first time it's needed (then cached for offline use).
PDF
pdf.js by Mozilla. Renders PDF pages, extracts text, and supports OCR scanning of image-only pages via Tesseract. Loaded only for PDF files.
3D / CAD
occt-import-js (OpenCASCADE). Tessellates STEP, IGES and BREP CAD solids to meshes for the in-browser 3D viewer. ~9 MB, loaded only when needed.
Archives
fflate for ZIP, libarchive.js for RAR / 7-Zip (CBR / CB7 comics), plus fzstd (Zstandard) and xzwasm (XZ / LZMA2) decompressors. Loaded on demand.
Databases
sql.js. SQLite compiled to WASM - lists tables, schema and row counts from .sqlite / .db files.
JPEG 2000
OpenJPEG compiled to WASM. Decodes JP2 / J2K images the browser can’t. Loaded only when needed.
PostScript
Ghostscript compiled to WASM. Renders EPS / PostScript graphics. ~16 MB, in the Complete download only.
Animation
lottie-web. Plays Lottie / Bodymovin JSON animations. Loaded only when needed.
File ID
Built-in identification for 1017+ file formats. Adobe, SolidWorks, Fusion 360, AutoCAD, Blender, Office, 3D mesh, executables, fonts, G-Code, log files, and more. Extracts version, author, and metadata from headers, ZIP manifests, ID3 tags, and gzipped XML projects. DOCX documents render with formatted text. AI-generation markers detected in photo metadata.
Install
Add to home screen for an offline PWA. Service worker caches the app shell. Use the buttons above to pre-cache heavy dependencies.
